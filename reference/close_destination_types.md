# Destination-type taxonomy

The destination-type taxonomy with leaf expansions (free). Use it to
look up the numeric `type` ids the data routes filter on (e.g. grocery,
restaurants).

## Usage

``` r
close_destination_types(client)
```

## Arguments

- client:

  A
  [`close_client()`](https://henryspatialanalysis.github.io/closecity-r/reference/close_client.md).

## Value

A
[close_reply](https://henryspatialanalysis.github.io/closecity-r/reference/close_reply.md).
